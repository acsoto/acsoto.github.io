MacOS全局可执行文件

发表于 2021-11-30 更新于 2021-12-01

sudo vim /usr/local/bin/command

/usr/local/bin/ 下创建的文件默认在path中
必须使用sudo

sudo chmod +x /usr/local/bin/command

即可全局使用 command
